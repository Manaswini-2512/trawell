# 🧭 Trawell – AI Travel & Cultural Marketplace

> An AI-powered travel platform that helps users discover cultural destinations, generate personalized travel itineraries, and explore local artisan products through a modern full-stack web application.

---

## 📌 Problem Statement

Modern travel platforms primarily focus on popular tourist destinations while overlooking local culture, artisans, and personalized travel planning. Travelers often struggle to discover authentic experiences while local craftsmen receive limited digital exposure.

**Trawell** addresses this challenge by combining AI-powered itinerary generation with a cultural marketplace that promotes sustainable tourism and supports local artisans.

---

# ✨ Key Features

### 🤖 AI-Powered Travel Planning

- Generate personalized travel itineraries using **Google Gemini**.
- Receive destination recommendations based on user preferences.
- Intelligent travel planning for unique travel experiences.

### 🌍 Cultural Destination Discovery

- Explore cultural landmarks and tourist attractions.
- Discover heritage locations and authentic local experiences.

### 🛍️ Artisan Marketplace

- Browse handcrafted products created by local artisans.
- Promote traditional crafts and support local communities.

### 💻 Modern User Experience

- Responsive interface built with React and Tailwind CSS.
- Modular React components for scalable development.
- Seamless navigation across Home, Destinations, Marketplace, and Contact pages.

# 🚀 Getting Started

## Prerequisites

- Node.js (v16 or above)
- npm
- Modern Web Browser

---

## Clone Repository

```bash
git clone https://github.com/Manaswini-2512/trawell.git

cd trawell
```

---

## Install Dependencies

```bash
npm install
```

---

## Run Development Server

```bash
npm run dev
```
